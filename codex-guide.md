## Codex 高频使用场景

### 1. 代码生成

Codex 可以帮助生成基础代码、接口逻辑、工具函数、脚本模板和常见业务模块。

### 2. Bug 排查

遇到报错时，可以结合错误日志、相关代码和调用链路，让 Codex 辅助分析可能原因。

### 3. 测试补全

Codex 可以辅助生成单元测试、边界条件测试和异常场景测试，降低补测试的时间成本。

### 4. 项目理解

接手老项目时，可以用 Codex 辅助阅读目录结构、函数关系和模块逻辑。

### 5. 文档整理

接口说明、部署文档、使用说明、变更记录，都可以借助 Codex 提高整理效率。
